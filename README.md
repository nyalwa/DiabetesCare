# Diabetes Management System

## Development Environment Specifications

| Component | Specification |
| :--- | :--- |
| **Operating System** | Windows 11 |
| **IDE** | Anti Gravity IDE (Google, built on VS Code) |
| **Python Version** | 3.12.4 |
| **Package Manager** | pip 26.0.1 |
| **Virtual Environment** | Python venv |

## Project Setup
The system is built using Flask and SQLAlchemy, with a virtual environment managed via `venv`.

### Key Files
- `app.py`: Main application logic.
- `database.py`: Database schema and configuration.
- `requirements.txt`: Project dependencies.
- `venv/`: Local virtual environment.
